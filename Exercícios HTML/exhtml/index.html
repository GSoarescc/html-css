Em HTML, a tag head é usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site

Body:

A tag Body engloba todos os elementos do corpo do documento

Símbolos e emojis

a tag & pode ser utilizada para criar diferentes símbolos
Alguns exemplos são:

  1. dollar : $
  2. euro: €
  3. male: ♂
  4. female: ♀
  5. copy: ©
  6. trade: ™

Emojis podem ser criados a partir de códigos começando com "&#x"
exemplo de emoji: 🏈

Listas:

Posso criar uma lista ordenada com a tag <ol></ol>
os itens da lista são criados com a tag li
  1. acordar
  2. tomar banho
  3. tomar café
  4. escovar os dentes
  5. estudar
é possível mudar o estilo de ordenação com o parâmetro type
  1. acordar
  2. tomar banho
  3. tomar café
  4. escovar os dentes
  5. estudar

também é possível criar listas sem ordenação com a tag <ul></ul>

  • acordar
  • tomar banho
  • tomar café
  • escovar os dentes
  • estudar

Links

links podem ser criados com a tag a

Alguns parêmetros utilizados são:

rel
Determina a relação do link com a página principal(External,prev,next, nofollow...)
target
Epecifica onde o link deve ser aberto(_blank,_self,_top)

links internos x externos

Link interno: Página 2 (leva para a segunda página do site)
link externo: Site do Flamengo

Links em outras pastas

Caso uma página esteja em uma pasta diferente, precisamos especificar no link. O atalho "Ctrl + Barra de espaço" faz isso automaticamente

Para voltar para uma página fora da pasta atual, usamos "../" na href
